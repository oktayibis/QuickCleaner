# Changelog

All notable changes to QuickCleaner will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-26

### Added

- Initial release
- Cache Cleaner for system and application caches
- Developer Tools cleaner (Xcode, npm, pip, Docker, etc.)
- Leftover Files scanner for orphaned app data
- Large Files finder with size sorting
- Duplicate Finder using hash-based detection
- System Info viewer
- DMG build script for easy distribution

### Features

- Native macOS app built with Swift & SwiftUI
- Modern, intuitive interface
- Safe file operations with confirmation dialogs
- Progress tracking for scan operations
